COMP 116: Introduction to Computer Security

Lab: Packet Sleuth

Due: Thursday, September 21st

This lab is worth 25 points. You must complete this lab individiually, on your own.

Objectives

  • Learn how to read and dissect files and sensitive information (including usernames and passwords) from packet sets using open source tools
  • Work with real network traffic from arguably the world's most dangerous computer network
  • Understand the dangers of sending data over an open network unencrypted, "in-the-clear"
  • Learn and utilize encoding schemes

Overview

You are given four sets of packet captures from two different networks in PCAP format to analyze: one from a network where file transfers occurred, one from a university network, and the others from arguably the world's most hostile network, DEF CON in Las Vegas, NV.

Instructions

Download the following PCAP files inside of your Kali Linux virtual machine:

  1. Set 1: http://www.cs.tufts.edu/comp/116/set1.pcap (5.6 MB)
  2. Set 2: http://www.cs.tufts.edu/comp/116/set2.pcap (64 MB. NOTE: I am not responsible for the contents in this PCAP.)
  3. Set 3: http://www.cs.tufts.edu/comp/116/set3.pcap (64 MB. NOTE: I am not responsible for the contents in this PCAP.)
  4. Set 4: http://www.cs.tufts.edu/comp/116/set4.pcap (487 KB)

Open and analyze the sets of data, one at a time, using Wireshark on Kali Linux. It is highly recommended that you also use additional open source tools (e.g., Ettercap, dsniff) for your analysis.
